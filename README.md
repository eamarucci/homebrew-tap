# Homebrew Tap

Repositorio de formulas Homebrew.

## Instalacao

```bash
brew tap eamarucci/tap
```

## Formulas disponiveis

| Formula | Descricao |
|---------|-----------|
| `ask` | CLI para consultar modelos de IA no terminal |

## Uso

```bash
# Instalar
brew install ask

# Atualizar
brew upgrade ask
```
